## About Me

Data Architect with **14+ years** of experience designing and operating enterprise data platforms — from legacy ETL pipelines and on-premise data warehouses to modern cloud-native architectures on AWS.

Currently at **ExamRoom.AI Corp** (Lake Mary, Florida), architecting and operating the full data engineering stack: Airflow 3 orchestration, dbt transformation layer (100+ models), Airbyte ingestion (20+ sources), OpenMetadata data catalog, Superset analytics, Grafana observability, and an agentic AI platform built on Ollama and MLflow — all running on AWS EKS.

| 2016 – 2024 | Altimetrik | Senior Architect — Data Engineering & Analytics |
| 2011 – 2016 | Virtusa | Associate Consultant — Data Warehousing |
